Mi experiencia con proyectos reales

Publicado el 3 de junio de 2025

Durante mi formación como estudiante de Ingeniería de Software en la UTP, he tenido la oportunidad de desarrollar varios proyectos reales que han sido clave para mejorar mis habilidades como programador web full stack.

Intranet para Librería

Este proyecto consistió en una aplicación web para la gestión de ventas y almacén de una librería. Implementé un sistema de login seguro, CRUD de productos y reportes. Utilicé tecnologías como HTML, PHP y MySQL.

SportFlexx: tienda de ropa online

En este proyecto diseñé una tienda de ropa con catálogo, carrito de compras, y una intranet de administración para gestionar productos. Me encargué del frontend en HTML, CSS y JavaScript, y de la estructura lógica de la intranet.

Lecciones aprendidas

  • Planificación del flujo de usuarios y estructura de la base de datos.
  • Comunicación efectiva con usuarios no técnicos.
  • Buenas prácticas de código, control de versiones con Git y debugging.

Estos proyectos no solo me han permitido aplicar lo aprendido en clases, sino también enfrentar problemas reales como errores en producción, cambios de requerimientos y necesidades de diseño responsive.

Estoy emocionado por seguir mejorando y desarrollando proyectos más complejos. ¡Gracias por leer!
